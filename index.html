What is Shiny?

Shiny is an R package that allows you to create web apps from R.

So?

You can create sign up forms

You can organize your recipes

You can automate annoyings tasks

You can predict if you will survive your visit to the known world

Many other things

Do I need a web server?

There are several ways to run a Shiny App:

* From within R (opens in a browser)

* In Rstudio (opens in Rstudio viewer)

* Shiny Server on a server

What else can I do with Shiny?

Connect to databases to store data

Connect to Google Docs to store data

Import and export files

Include HTML files

Inclue maps (leaflet, GoogleVis, rMaps)

Include interactive visualizations (ggvis)
